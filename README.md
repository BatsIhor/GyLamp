# GyLaml

GyLaml - Приложение для управления лампой [GyverLamp](https://github.com/AlexGyver/GyverLamp) 

## Как собрать

```
sudo gem install cocoapods
git clone https://github.com/nikit6000/GyLamp.git && cd GyLamp
pod install
```
